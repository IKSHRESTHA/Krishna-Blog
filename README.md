## Local Development
1. Install [Quarto](https://quarto.org/docs/get-started/).
2. Clone this repository:
   ```
   git clone https://github.com/IKSHRESTHA/Krishna-Blog.git
   cd Krishna-Blog
   ```
3. Render the site locally:
   ```
   quarto preview
   ```
4. To manage R dependencies, use `renv`:
   ```R
   source("renv-init.R")
   # Install packages as needed, then
   renv::snapshot()
   ```

## Deployment

- After pushing to `main`, the site will be published at: https://ikshrestha.github.io/Krishna-Blog/
